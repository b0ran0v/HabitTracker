# HabitTracker

HabitTracker is a .NET Android application designed to help users track and manage their daily habits.

## Features

*   **Habit Tracking**: Monitor your daily habits and progress.
*   **Habit Management**: View and manage your list of habits.
*   **Local Storage**: Data is persisted locally using SQLite.

## Technologies Used

*   **Framework**: .NET for Android (net10.0-android)
*   **Database**: SQLite (sqlite-net-pcl)
*   **UI**: Android XML Layouts with Material Design components

## Getting Started

1.  Clone the repository.
2.  Open the solution in Visual Studio or your preferred .NET IDE.
3.  Build and run the project on an Android emulator or physical device.
